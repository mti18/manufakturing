### Specification

- `Laravel 9.19`
- `VueJS 3.2.41`
- `Metronic 8`

### Installation

Installation is simple. It can be install using the following command:
```sh
$ git clone https://gitlab.com/McFlyon/latest-base-mti.git 
$ cd latest-base-mti
$ composer install
$ npm install
```

Optional
```sh
& php artisan storage:link
```

Running the application: (using 2 terminal)
```sh
$ npm run dev 
$ php artisan serve
```
